# rails-react-api
Creating a restful api with rails and react


# Test locally
1. Clone it
2. cd into the project then into backend
3. Run `bundle`
4. Run `rails db:migrate:reset`
5. Run `rails db:seed`
6. Run `foreman start -p 3000 `
